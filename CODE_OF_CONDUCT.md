# Code of conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1. A full copy of the text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Report unacceptable behavior to the repository maintainers via GitHub issues or, if you prefer a private channel, the contact methods listed on the maintainers’ GitHub profiles.

Maintainers will review and respond in good faith. Participants who do not follow this code may be temporarily or permanently excluded from project spaces.
